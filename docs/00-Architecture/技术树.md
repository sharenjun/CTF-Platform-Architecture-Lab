# 技术树

## 技术分层

| 层级 | 关注点 |
| --- | --- |
| 前端层 | 页面、路由、状态管理、接口调用 |
| 接入层 | 网关、鉴权、限流、跨域 |
| 应用层 | 用户、比赛、题目、提交等业务服务 |
| 异步层 | 消息队列、判题任务、排行榜刷新 |
| 基础设施层 | 数据库、缓存、对象存储、Docker |
| 运维层 | 日志、监控、告警、部署流水线 |

## 后端技术点

- Spring Boot 应用开发
- Spring Security 鉴权授权
- MyBatis 或 JPA 数据访问
- Redis 缓存与分布式锁
- 消息队列异步判题
- CompletableFuture 并发编排
- Docker Java API 容器控制

## 前端技术点

- 登录态管理
- 比赛列表和详情页
- 题目详情和提交表单
- 排行榜实时或准实时刷新
- 管理后台表单与列表

## 数据存储

- MySQL：核心业务数据。
- Redis：会话、缓存、排行榜、限流。
- 对象存储或本地文件：题目附件、导入导出文件。

## 平台工程能力

- 单元测试和集成测试
- API 文档
- Docker Compose 本地开发环境
- CI 构建
- 日志追踪和问题定位
