# 知识树

## CTF 平台知识域

本文用于整理开发 CTF 平台时需要掌握的业务知识、Java 知识和基础设施知识。

## 业务知识

- CTF 比赛模式：Jeopardy、Attack Defense、混合模式。
- 题目类型：Web、Pwn、Reverse、Crypto、Misc、Forensics。
- 计分方式：固定分、动态分、一血奖励、时间惩罚。
- 排行榜规则：个人榜、团队榜、比赛内榜、全站榜。
- 反作弊：提交频率限制、异常行为审计、环境隔离。
