# 业务树

## 平台业务总览

CTF 平台围绕比赛组织、选手答题、自动判题和运维管理展开。

## 用户侧业务

- 注册与登录
- 个人资料维护
- 团队创建与加入
- 比赛报名
- 题目浏览
- 动态环境申请
- 答案提交
- 提交记录查询
- 排行榜查看

## 管理侧业务

- 用户管理
- 角色与权限管理
- 比赛创建与配置
- 题目管理
- 附件与镜像管理
- 公告管理
- 提交记录审计
- 容器实例管理
- 平台运行监控

## 核心业务对象

| 对象 | 说明 |
| --- | --- |
| User | 平台用户 |
| Team | 参赛团队 |
| Role | 权限角色 |
| Contest | 比赛 |
| Challenge | 题目 |
| Submission | 提交记录 |
| JudgeTask | 判题任务 |
| ContainerInstance | 动态靶机实例 |
| Scoreboard | 排行榜视图 |

## 业务边界

- 用户系统负责身份和权限，不直接处理判题逻辑。
- 比赛系统负责题目、提交和积分，不直接操作底层容器。
- 判题系统负责判断答案正确性，不承担用户会话管理。
- 容器系统负责运行环境生命周期，不决定比赛分数。
